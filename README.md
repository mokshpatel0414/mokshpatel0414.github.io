# mokshpatel0414.github.io

My personal portfolio — a single-page site showcasing my data analytics work. Built from scratch with no frameworks, no build step, and no dependencies. Just one HTML file.

**Live at [mokshpatel0414.github.io](https://mokshpatel0414.github.io)**

## About

I'm Moksh Patel, a Computer Science graduate from Cal State Fullerton and a data analyst. This site pulls together my four main projects, the findings behind them, and ways to get in touch. The design is deliberately minimal — dark, quiet, and text-forward — because I'd rather let the work speak than dress it up.

## Featured projects

- **Spotify Charts** — SQL analysis of 26.2M chart entries across 70+ countries, built into an interactive Streamlit dashboard
- **NBA Three-Point Revolution** — multi-CTE SQL analysis of 65,000+ games across 22 seasons
- **Predicting COVID-19 Deaths** — a Python regression pipeline forecasting mortality two weeks ahead across 159 countries
- **Bike Sales Dashboard** — Excel pivot-table analysis and interactive dashboard over 1,000+ customer records

## Built with

- Hand-written HTML and CSS, no frameworks
- Fraunces and JetBrains Mono via Google Fonts
- Vanilla JavaScript for scroll-reveal animations and the sticky nav
- Hosted on GitHub Pages

## Running it locally

No build step needed. Clone the repo and open index.html in any browser:

    git clone https://github.com/mokshpatel0414/mokshpatel0414.github.io.git
    cd mokshpatel0414.github.io
    open index.html
