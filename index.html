02 Projects
Spotify Charts — A Cross-Market Analysis
GitHub ↗
DuckDBSQLStreamlitPlotly

SQL queries against 26.2M Spotify chart entries from 70+ countries, using chained CTEs, window functions, and conditional aggregation to surface cross-market consumption patterns. Built into an interactive Streamlit dashboard exploring 54 markets.

Time to reach #1 on the US chart fell 75% — from 22.6 to 5.7 days between 2017 and 2021 — while chart lifespan halved from 452 to 199 days.

NBA Three-Point Revolution — A SQL Analysis
GitHub ↗
SQLiteSQLWindow Functions

Multi-CTE SQL queries against a 2.5GB database of 65,000+ NBA games across 22 seasons, quantifying how 3-point shooting reshaped the league. Caught a missing 2012-13 regular season through data-integrity checks.

The win-rate edge of high-volume 3-point teams peaked at +9.1 points in the late 2000s, then decayed to near-zero by 2022 as the strategy commoditized across all 30 teams.

Predicting COVID-19 Deaths — A Python Regression Analysis
GitHub ↗
Pythonstatsmodelsscikit-learnpandas

An end-to-end pipeline merging 267K country-day records from Our World in Data with World Bank indicators across 159 countries, building a 14-day-ahead mortality target. Five regression models evaluated on a held-out 2023 test set.

The top-performing models scored only 2,372 of 56,495 test rows due to missing data — exposing a reporting bias toward wealthy nations and showing why a single-variable baseline generalized better worldwide.

Bike Sales Dashboard — An Excel Analysis
GitHub ↗
ExcelPivot TablesDashboards

Analysis of 1,000+ customer records using pivot tables to surface purchasing patterns across demographics and commute distance, built into an interactive dashboard with 10+ slicers for dynamic filtering by marital status, region, and education.

Identified middle-aged, short-distance commuters as the highest-probability customer segment, pointing toward targeted marketing and more efficient customer acquisition.
